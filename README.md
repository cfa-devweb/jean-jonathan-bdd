### Le MCD 

En premier lieu nous avons réalisé un Model Conceptuel des Données sur Lucidchart pour nous permettre de visualiser les différentes tables et contraintes qui
nous ont été imposées.

### Le MLD 

Ici la réalitation d'un Model Logique de Données sous forme de shéma était facultatif pour la compréhension des relations entre les tables.


### Le MPD

Grace au MCD nous avons créer un fichier SQL pour notre model physique des données, par la suite nous avons compris qu'il fallait effectuer quelques changements
par rapport a notre MCD quand au nommage de nos colonnes.Chose qui s'est avéré encore plus véridict lors de la création de nos script de créations et d'insertions.
